# 🍽️ Ma Table

> Planificateur de repas en famille — PWA & Android

**Ma Table** est une application web progressive (PWA) pour planifier les repas en famille, gérer une liste de courses intelligente et conserver un carnet de recettes, le tout synchronisé en temps réel entre les membres du foyer.

## 🔗 Liens

- **Version web** : [ma-table-69581.web.app](https://ma-table-69581.web.app)
- **Télécharger l'APK Android** : [Dernière version](../../releases/latest/download/ma-table.apk)

## ✨ Fonctionnalités principales

- 📅 Calendrier des repas (semaine / quinzaine)
- 🛒 Liste de courses avec autocomplétion et produits de saison
- 📖 Carnet de recettes avec export PDF
- 👨‍👩‍👧 Synchronisation temps réel entre membres du foyer
- 🔔 Notifications de rappel avant les repas
- 🌙 Thème clair / sombre
- 📵 Mode hors-ligne (PWA)
